
当程序创建 ApplicationContext 实例时，通常也是以 Resource 的方式来访问配置文件的，所以 ApplicationContext 完全支持
ClassPathResource、FileSystemResource、ServletContextResource 等资源访问方式。

ApplicationContext 实现类可以指定访问策略，具体如下：

ClassPathXmlApplicationContext：对应使用 ClassPathResource 进行资源访问。
FileSystemApplicationContext：对应使用 FileSystemResource 进行资源访问。
XmlWebApplicationContext：对应使用 ServletContextResource 进行资源访问。
